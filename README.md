# 方直智胜官网

## 项目说明

这是一个基于 `Vite + React + TypeScript` 的官网项目。

主要目录：

- `index.html`：Vite HTML 入口
- `src/`：React 页面、组件、状态和类型
- `assets/`：素材说明文件
- `test/`：React 页面迁移与路由测试

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run dev
```

构建生产包：

```bash
npm run build
```

预览生产包：

```bash
npm run preview
```

## 测试

运行测试：

```bash
npm test
```
